

Haga un algoritmo que solicite los lados de la cancha y 
calcule el área de la misma, para eso tenga en cuenta 
las siguientes indicaciones:

1. Realice el proceso mediante programación estructurada 
(secuencial de pasos, no orientado a objetos ni funciones)
2. Modifique el ejercicio y ahora haga uso de 2 funciones, 
una para solicitar los datos y otra para calcular el área.
    1. ingresarDatos(mensaje): Esta función recibirá 
    un mensaje solicitando el ingreso del dato y deberá retornar 
    el valor ingresado de tipo numérico. 
    (la función será llamada cada vez que se ingrese un lado)
    2. calcularArea(ladoA,ladoB): esta función será la 
    encargada de realizar el proceso del calculo del área.
3. Modifique el ejercicio anterior y ahora creará la clase 
"Cancha" la cual tendrá los atributos ladoA y ladoB, 
así como los métodos creados para el ingreso de datos y 
el calculo del área, deberá crear 3 objetos de tipo Cancha 
con el fin de calcular su área. (usted determina si crea 
o no el método constructor).
